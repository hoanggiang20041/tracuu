🔄 Test Cross-Browser Sync

Hướng dẫn test đồng bộ 2FA giữa các trình duyệt trên localhost

📋 Bước 1: Mở 2 trình duyệt

Mở 2 trình duyệt khác nhau (Chrome, Firefox, Edge, etc.) và truy cập:

http://127.0.0.1:5500/security.html

🔐 Bước 2: Đăng nhập cả 2 trình duyệt

Đăng nhập với cùng 1 tài khoản admin:

Username: admin
Password: admin123

⚙️ Bước 3: Bật 2FA ở trình duyệt A

Trong trình duyệt A:

  1. Vào tab "2FA"
  2. Nhấn "Kích hoạt lại 2FA"
  3. Nhập mã từ Google Authenticator
  4. Nhấn "Xác nhận"

🔄 Bước 4: Sync ở trình duyệt B

Trong trình duyệt B:

  1. Vào tab "2FA"
  2. Nhấn "Manual Sync" (nút xanh lá)
  3. Kiểm tra trạng thái 2FA

🧪 Bước 5: Test đăng nhập

Mở trình duyệt thứ 3 và test đăng nhập:

  1. Mở login.html
  2. Đăng nhập với admin123
  3. Kiểm tra có yêu cầu 2FA không

⚠️ Lưu ý quan trọng

Vấn đề localhost: Mỗi trình duyệt có localStorage riêng biệt trên localhost. Để sync thực sự, cần:

  • Sử dụng "Manual Sync" button
  • Hoặc deploy lên web server thực
  • Hoặc sử dụng cùng 1 trình duyệt với nhiều tab

✅ Giải pháp tạm thời

Để test đầy đủ, hãy:

  1. Mở cùng 1 trình duyệt với 2 tab
  2. Tab 1: Bật 2FA
  3. Tab 2: Refresh → Sẽ tự động sync
  4. Mở tab 3: Test đăng nhập
