🔄 Hướng dẫn Sync 2FA giữa các trình duyệt

Giải pháp để đồng bộ 2FA giữa Chrome, Firefox, Edge, Safari...

🎯 Vấn đề hiện tại

Mỗi trình duyệt có localStorage riêng biệt trên localhost, nên không thể sync tự động. Cần sync thủ công.

✅ Giải pháp: Export/Import 2FA Data

Đã thêm 2 button mới trong trang security:

📋 Cách thực hiện

Bước 1: Export từ trình duyệt đã bật 2FA

  1. Mở trình duyệt đã bật 2FA (Chrome)
  2. Vào security.html
  3. Tab "2FA" → Nhấn "Export 2FA Data"
  4. Copy data trong modal

Bước 2: Import vào trình duyệt khác

  1. Mở trình duyệt khác (Firefox)
  2. Vào security.html
  3. Tab "2FA" → Nhấn "Import 2FA Data"
  4. Paste data đã copy
  5. Nhấn "Import Data"

Bước 3: Test đăng nhập

  1. Mở login.html
  2. Đăng nhập với admin123
  3. Kiểm tra có yêu cầu 2FA không

⚠️ Lưu ý quan trọng

  • Chỉ sync 1 lần: Sau khi import, 2FA sẽ hoạt động bình thường
  • Secret key cố định: Cùng 1 secret key cho tất cả trình duyệt
  • Google Authenticator: Sử dụng cùng 1 app cho tất cả trình duyệt
  • Bảo mật: Data export chứa secret key, cần bảo mật

🚀 Khi deploy lên web thật

Khi deploy lên server thật, localStorage sẽ sync tự động giữa các trình duyệt vì:

  • Cùng domain
  • Cùng storage scope
  • Auto sync sẽ hoạt động

📱 Cách thêm vào Google Authenticator

  1. Mở Google Authenticator app
  2. Nhấn "+" để thêm account
  3. Chọn "Enter a setup key"
  4. Nhập secret key từ "Xem Secret Key"
  5. Nhập tên: "Admin 2FA"
  6. Nhấn "Add"
